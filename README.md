Use this to score your content against a set of configurable editorial principles.

Enter the URL of the content, which is fetched via the Anthropic API server-side.

Select Edit to change the principles. A default set is provided for your convenience, but you can edit, add or rank the principles if you wish to change them.

Each article is scored on a scale, with commentary on each dimension provided alongside the overall score.

View the scoring scale reference card — shown on the page at all times, so anyone using it can see the bands. Or edit either the text or numbers to fit your needs.
