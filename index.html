Setup: Cloudflare Worker (one-time, ~5 minutes)
  1. Go to workers.cloudflare.com and create a free account
  2. Click Create application → Create Worker → give it a name
  3. Click Edit code, delete everything, and paste the contents of worker.js (download below)
  4. Click Deploy
  5. Go to Settings → Variables → Add variable — name it ANTHROPIC_API_KEY, paste your API key as the value, click Encrypt then Save
  6. Copy your Worker URL (shown at the top, like https://my-scorer.yourname.workers.dev) and paste it into the Worker URL field above
