Setup: Cloudflare Worker (one-time, ~5 minutes)
  1. Go to workers.cloudflare.com and create a free account
  2. Click Create application → Hello World → give it a name → Deploy
  3. Click Edit code, delete everything, paste the contents of worker.js → Deploy
  4. Go to Settings → Variables and Secrets → Add → name: ANTHROPIC_API_KEY → paste your key → Deploy
  5. Copy your Worker URL and paste it into the Worker URL field above
